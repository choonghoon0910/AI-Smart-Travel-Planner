# ✈️ AI Smart Travel Planner

**AI Smart Travel Planner**는 사용자의 **선호도**를 기반으로  
최적의 **항공, 관광지, 숙소**를 단계별로 추천하고,  
선택된 항목으로 **최종 여행 일정을 구성**하는 **모바일 기반 지능형 여행 계획 앱**입니다.

---

## 📌 프로젝트 개요

> “AI의 추천과 사용자의 선택을 결합하여, 실현 가능한 여행을 완성하는 어플”

많은 여행자들이 복잡한 검색 과정과 예산 문제로 인해  
여행 계획 단계에서 어려움을 겪습니다.  
본 프로젝트는 이러한 문제를 해결하기 위해  
**사용자 선호도 선택 →  AI 추천 → DB 저장 → 최종 일정 구성**이라는 구조를 통해  
여행 계획 과정을 단순화하고, 데이터 무결성을 확보하여 완성도 높은 여행 일정을 제공합니다.

## 🧩 주요 기능 (MVP 기준)

| 기능 구분 | 설명 |
|------------|------|
| 🎯 **다단계 정보 입력** | 4단계 설문(기본 정보 → 여행 스타일 → AI 추천 정보 선택 → 완성된 플랜 확인)을 통해 사용자 데이터를 수집 |
| ✈️ **순차적 추천 및 선택** | 항공권 → 관광지 → 숙소 순으로 AI가 추천 목록을 제시하고 사용자가 선택 |
| 💾 **DB 기반 상태 저장** | 사용자의 모든 선택 내역을 **프로젝트 ID 기준으로 H2 DB에 즉시 갱신(UPDATE)** |
| 📈 **API 연동 파이프라인** | 백엔드(Spring)가 AI 서버(Python)에 요청을 중개하고, JSON 응답을 DTO로 변환 |
| 🏁 **최종 계획 요약** | 선택된 항공편, 숙소, 관광지 목록을 최종 화면에 표시하여 프로젝트 완료 |
| 🔒 **사용자 인증** | 이메일 및 비밀번호 기반의 회원가입 / 로그인 기능 구현 |

---

## 🧠 시스템 구조
  [Android Client] ⇄ [Spring Boot Server] ⇄ [H2 Database]
                        ↑
      [AI Server (Python)] ⇄ [Recommendation Model]

- **Retrofit**: 안드로이드 → 백엔드 간 JSON 기반 통신  
- **Spring Boot**: 요청을 받아 AI 서버에 전달 후 응답을 DTO로 변환  
- **H2 Database**: 프로젝트별 단계 진행 상황 및 선택 데이터 저장
